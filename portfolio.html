I have a coded a few projects. Below are some of my best and most pround works.
The projects are coded in programming languages, such as Python, HTML, and CSS.

Tic Tac Toe

Coded this basic python Tic-Tac-Toe game using the tkinter library for the GUI.
It allows two players to take turns and checks for wins or ties, and provides
a "Play Again" and "Exit" option through a menu bar.

Snake

Coded this classic snake game programmed in Python using the Turtle module.
The player uses the arrow keys to change the direction of the snake.
The score is displayed at the top of the screen, and if the player hits the wall
or the snake's body, the game restarts.

Weather App

Coded this weather app using HTML, CSS, and JavaScript. I utilised
OpenWeatherApp API to gather and display information on different cities'
temperatures.

Calculator

I coded this basic arithmetic GUI calculator in Python. It allows users to perform
